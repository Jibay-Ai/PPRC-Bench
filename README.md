Persian Reasoning, Comprehension, Conversation Benchmark

A benchmark developed by Jibay Ai for evaluating Persian language understanding and execution capabilities in modern AI systems.

---

About

PRCC-BENCH is a Persian benchmark designed to evaluate how effectively language models understand, reason, analyze, follow instructions, answer questions, and extract information.

Unlike benchmarks focused only on factual recall, PRCC-BENCH emphasizes:

- Understanding
- Reasoning
- Execution
- Reliability
- Consistency
- Language intelligence

This benchmark is designed to simulate practical and realistic language tasks.

Evaluation Categories

1. Comprehension

Measures whether a model truly understands:

- Context
- Events
- Subject references
- Object tracking
- Multi-sentence understanding
- Temporal relations
- Meaning consistency

Abilities measured:

✓ Read and understand text
✓ Track entities
✓ Follow narratives
✓ Detect hidden references

---

2. Reasoning

Measures logical processing:

- Multi-step reasoning
- Deduction
- Cause and effect
- Decision making
- Temporal understanding
- Context dependency

Abilities measured:

✓ Infer conclusions
✓ Solve contextual logic
✓ Maintain consistency

---

3. Question Answering

Evaluates:

- Correctness
- Precision
- Completeness
- Context awareness

Question types include:

- Direct QA
- Contextual QA
- Multi-part QA
- Implicit answers

Abilities measured:

✓ Accurate responses
✓ Context preservation

---

4. Information Extraction

Measures:

- Entity extraction
- Fact retrieval
- Number extraction
- Relation understanding
- Context filtering

Abilities measured:

✓ Extract useful information
✓ Ignore irrelevant details

---

5. Instruction Following

Measures:

- Constraint adherence
- Exact formatting
- Rule execution
- Response discipline

Abilities measured:

✓ Execute instructions
✓ Preserve requested structure

---

6. Analysis

Measures:

- Problem decomposition
- Evidence comparison
- Deep understanding
- Analytical thinking

Abilities measured:

✓ Draw conclusions

Scoring System

Result| Score
Correct| +5
Incorrect| 0
Blank| 0
Near Match| 0

Rule:

Only fully correct responses receive points.

---

Judging Modes

Exact Match

Normalization:

- Spaces
- Punctuation
- Persian digits

Comparison occurs after normalization.

---

Semantic Match

Answers are accepted only when meaning remains fully equivalent.

---

List Exact

All requested facts must appear.

---

Order Exact

All required outputs must remain in requested order.
